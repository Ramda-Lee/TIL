# LoadBalancer

- `upstream` 블록으로 백엔드 풀 구성
    
    ```bash
                           ┌──► [backend1 :8080]
    [client] ──► [nginx]  ─┤
                           └──► [backend2 :8080]
    ```
    
    - 백엔드 컨트롤러 수정
        
        ```bash
        package com.example.backend;
        
        import jakarta.servlet.http.HttpServletRequest;
        import org.springframework.beans.factory.annotation.Value;
        import org.springframework.web.bind.annotation.GetMapping;
        import org.springframework.web.bind.annotation.RestController;
        
        import java.util.LinkedHashMap;
        import java.util.Map;
        
        @RestController
        public class IpTestController {
        
            @Value("${SERVER_NAME:unknown}")
            private String serverName;
        
            @GetMapping("/nginx")
            public Map<String, String> nginxTest(HttpServletRequest request) {
                Map<String, String> result = new LinkedHashMap<>();
                result.put("server", serverName);                  // ★ 추가
                result.put("remoteAddr", request.getRemoteAddr());
                result.put("X-Forwarded-For", request.getHeader("X-Forwarded-For"));
                result.put("X-Real-IP", request.getHeader("X-Real-IP"));
                result.put("Host", request.getHeader("Host"));
                return result;
            }
        }
        ```
        
    - docker-compose.yml
        
        ```bash
        services:
          backend1:
            build: ./backend
            expose:
              - "8080"
            environment:
              - SERVER_NAME=backend1
        
          backend2:
            build: ./backend
            expose:
              - "8080"
            environment:
              - SERVER_NAME=backend2
        
          nginx:
            image: nginx:1.27-alpine
            ports:
              - "8000:80"
            volumes:
              - ./nginx/nginx.conf:/etc/nginx/nginx.conf:ro
            depends_on:
              - backend1
              - backend2
        
          client:
            image: curlimages/curl:latest
            command: sleep infinity
            depends_on:
              - nginx
        ```
        
    - nginx.conf — upstream 블록 도입
        
        ```bash
        events {}
        
        http {
            # 백엔드 풀 정의
            upstream backend_pool {
                server backend1:8080;
                server backend2:8080;
            }
        
            server {
                listen 80;
        
                location / {
                    proxy_pass http://backend_pool;   # 단일 서버 → 풀로 변경
        
                    proxy_set_header Host              $host;
                    proxy_set_header X-Real-IP         $remote_addr;
                    proxy_set_header X-Forwarded-For   $proxy_add_x_forwarded_for;
                    proxy_set_header X-Forwarded-Proto $scheme;
                }
            }
        }
        ```
        
    - 실행
        
        ```bash
        docker compose down       # STEP 2 컨테이너 정리
        docker compose up --build
        ```
        
- Round Robin (기본) — 순서대로 분배
    - test

        ```bash
        done
        ```

- Weighted Round Robin (`weight=`) — 서버 성능 차이 반영
    - nginx.conf 수정
        
        ```bash
        upstream backend_pool {
            server backend1:8080 weight=3;
            server backend2:8080 weight=1;
        }
        ```
        
    - 설정 반영 및 테스트
        
        ```bash
        docker compose restart nginx
        
        for i in $(seq 1 8); do
          curl -s http://localhost:8000/nginx | grep server
        done
        ```

- Least Connections (`least_conn`) — 커넥션 적은 곳으로
    - nginx.conf 수정 및 실행
        
        ```bash
        upstream backend_pool {
            least_conn;                       # ★ 추가
            server backend1:8080;
            server backend2:8080;
        }
        
        docker compose restart nginx
        ```
        
    - 컨트롤러 추가
        
        ```bash
        @GetMapping("/slow")
        public Map<String, String> slow(HttpServletRequest request) throws InterruptedException {
            Thread.sleep(3000);   // 3초 대기
            Map<String, String> result = new LinkedHashMap<>();
            result.put("server", serverName);
            return result;
        }
        ```
