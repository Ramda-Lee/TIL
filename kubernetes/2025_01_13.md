## Kubernetes Pod 실습

### Pod 조회
- 전체 목록 간단 조회
```
  kubectl get pods
```
- 전체 목록 상세 조회
```
  kubectl get pods -o wide
```
- Pod 상세조회 (내부 컨테이너 종류 및 에러 등 정보가 나와 있다.)
```
  kubectl describe pod [pod 이름]
```

### Pod 생성
- 명령어로 생성하기
```
  kubectl run [pod 이름] --image=[image 이름]
```
- yaml 사용해서 생성하기
```
  kubectl run [pod 이름] --image=[image 이름] --dry-run=client -o yaml > [yaml 이름] # yaml 파일 생성해준다
  kubectl create -f [yaml 이름] # yaml을 이용하여 pod 생성
  kubectl apply -f [yaml 이름]  # -f는 파일이름 또는 경로를 지정한다는 의미, yaml 파일 수정 후 적용
```

### Pod 삭제
```
  kubectl delete pod [pod 이름]
```
